# 🎬 Netflix Content Analytics Dashboard

> **A recruiter-ready Power BI project by Shaik Hafeefa**

## 📌 Overview
This project analyzes Netflix's content library using Power BI to uncover trends in movies and TV shows. It demonstrates the complete analytics workflow including data cleaning, transformation, modeling, DAX calculations, and interactive dashboard design.

## 🚀 Objectives
- Analyze Netflix content distribution
- Compare Movies vs TV Shows
- Explore release year trends
- Analyze ratings, genres, and countries
- Build an interactive executive dashboard

## 🛠️ Tech Stack
- Power BI
- Power Query
- DAX
- Microsoft Excel / CSV
- Data Visualization

## 📊 Dashboard Features
- KPI Cards
- Interactive Slicers
- Genre Analysis
- Rating Distribution
- Release Year Trends
- Country-wise Analysis
- Movie vs TV Show Breakdown
