# Netflix Power BI Dashboard

## Overview

This project presents an interactive Netflix Dashboard built using Power BI. The dashboard provides insights into Netflix's content library, including Movies, TV Shows, ratings, genres, release years, and country-wise distribution.

## Dashboard Preview

(![Netflix Dashboard](Images/netflix-dashboard.png)

---

## Project Objectives

* Analyze Netflix content distribution.
* Compare Movies and TV Shows.
* Identify popular ratings and genres.
* Explore content trends over time.
* Visualize country-wise content availability.

## Insights

* Movies account for the majority of Netflix content.
* TV Shows represent a smaller but significant portion of the catalog.
* Content production increased significantly after 2015.
* Certain countries contribute a large share of Netflix titles.
